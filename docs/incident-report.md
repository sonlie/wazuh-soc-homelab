# Incident Report

## Summary

A reconnaissance activity was detected against the Ubuntu Server endpoint within the SOC lab environment.

The activity was generated from a Kali Linux machine using Nmap service discovery scanning.

The event was detected by Suricata IDS and forwarded to the Wazuh SIEM platform for analysis.

### Severity

Low

---

## Analysis

The attacker performed reconnaissance against the target system using Nmap.

Reconnaissance is commonly used during the initial stages of an attack to identify:

* Open ports
* Running services
* Service versions
* Potential vulnerabilities

Suricata detected HTTP requests generated by the Nmap Scripting Engine and generated an alert.

No successful exploitation was observed.

---

## Impact

No system compromise occurred.

The activity was limited to information gathering.
